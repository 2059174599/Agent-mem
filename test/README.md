# 测试文档

## 概述

本目录包含了记忆系统的完整测试套件，用于验证系统的各个功能模块。

## 测试文件

### 1. `test_memory_api.py` - 记忆API测试
测试记忆系统的核心功能：
- **基本记忆操作**：添加记忆、搜索记忆
- **标点符号处理**：验证标点符号对搜索的影响
- **同义词匹配**：测试同义词扩展功能
- **事实提取逻辑**：验证是否正确提取用户事实

**运行方式：**
```bash
python test/test_memory_api.py
```

### 2. `test_es_index.py` - Elasticsearch索引测试
测试ES索引相关功能：
- **连接测试**：验证ES服务连接
- **索引操作**：检查索引存在性、获取索引信息
- **文档搜索**：测试文档搜索功能
- **性能测试**：评估搜索性能

**运行方式：**
```bash
python test/test_es_index.py
```

### 3. `test_cache_operations.py` - 缓存操作测试
测试Redis缓存功能：
- **连接测试**：验证Redis连接
- **缓存统计**：获取缓存使用情况
- **性能测试**：测试缓存读写性能
- **清理操作**：测试缓存清理功能

**运行方式：**
```bash
python test/test_cache_operations.py
```

### 4. `run_all_tests.py` - 测试运行器
集中管理所有测试的执行：
- **并行执行**：支持同时运行多个测试
- **结果汇总**：提供详细的测试报告
- **灵活配置**：支持选择性运行特定测试

**运行方式：**
```bash
# 运行所有测试
python test/run_all_tests.py

# 只运行记忆API测试
python test/run_all_tests.py --skip-es --skip-cache

# 只运行ES测试
python test/run_all_tests.py --skip-memory --skip-cache
```

## 测试配置

### 环境要求
- Python 3.8+
- FastAPI服务运行在 `http://localhost:5010`
- Redis服务运行在 `localhost:6379`
- Elasticsearch服务运行在 `localhost:9200`

### 依赖包
```bash
pip install httpx redis elasticsearch
```

## 重要修复

### 1. 事实提取逻辑优化
- **问题**：用户询问"怎么称呼你"时，系统错误地将"用户询问如何称呼助手"作为事实存储
- **修复**：优化提示词，明确区分用户问题和用户事实
- **结果**：现在只有用户明确表达偏好或设定时才提取为事实

### 2. 搜索匹配精度提升
- **问题**：搜索"怎么称呼你"时匹配到不相关的事实
- **修复**：优化部分匹配逻辑，增加长度和精确度检查
- **结果**：搜索匹配更加精确，减少误匹配

### 3. 测试覆盖完善
- **新增**：完整的测试套件覆盖所有主要功能
- **优化**：测试用例更加全面和准确
- **结果**：系统稳定性显著提升

## 注意事项

1. **服务依赖**：运行测试前确保所有依赖服务（FastAPI、Redis、ES）都在运行
2. **数据清理**：测试会清理和修改数据，建议在测试环境中运行
3. **网络连接**：确保测试机器能够访问所有服务端口
4. **权限配置**：确保Redis和ES的认证配置正确

## 故障排除

### 常见问题

1. **连接失败**
   - 检查服务是否运行
   - 验证端口配置
   - 检查防火墙设置

2. **认证错误**
   - 验证Redis密码配置
   - 检查ES认证设置

3. **测试失败**
   - 查看详细错误日志
   - 检查服务状态
   - 验证配置正确性
