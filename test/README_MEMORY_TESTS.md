# 记忆功能测试用例说明

## 概述

本目录包含完整的记忆功能测试用例，重点测试：
1. 同一用户或代理ID添加的事实记忆，进行记忆搜索时是否能准确返回
2. 添加的记忆，下次进行相关问题，是否能召回相似问答

## 测试文件说明

### 1. `test_memory_quick.py` - 快速测试
- **功能**: 快速验证基本记忆添加和搜索功能
- **特点**: 轻量级，适合快速验证
- **测试内容**: 
  - 基本记忆添加
  - 记忆搜索
  - 用户代理隔离

### 2. `test_similar_qa_recall.py` - 相似问答召回测试
- **功能**: 专门测试相似问题的召回效果
- **特点**: 重点测试语义相似性
- **测试内容**:
  - 食物偏好相关问答
  - 爱好活动相关问答
  - 工作学习相关问答
  - 语义相似性测试

### 3. `test_memory_add_and_search.py` - 综合记忆测试
- **功能**: 完整的记忆功能测试套件
- **特点**: 全面测试各种场景
- **测试内容**:
  - 基本记忆流程
  - 用户代理隔离
  - 相似问题召回
  - 事实提取搜索
  - 记忆持久化

### 4. `test_search_accuracy.py` - 搜索准确性测试
- **功能**: 测试搜索结果的准确性
- **特点**: 验证搜索算法优化效果
- **测试内容**:
  - 不同查询的搜索效果
  - 相关性分析
  - 准确性评估

### 5. `test_recent_chats_real_data.py` - 最近对话测试
- **功能**: 使用真实数据测试最近对话功能
- **特点**: 验证chat_id修复效果
- **测试内容**:
  - ES数据获取
  - chat_id正确性
  - 最近对话功能

### 6. `run_memory_tests.py` - 测试运行器
- **功能**: 统一运行所有测试
- **特点**: 提供多种测试选项
- **使用方法**:
  ```bash
  python test/run_memory_tests.py quick     # 快速测试
  python test/run_memory_tests.py similar   # 相似问答召回测试
  python test/run_memory_tests.py full      # 综合记忆测试
  python test/run_memory_tests.py all       # 所有测试
  ```

## 测试场景

### 场景1: 基本记忆添加和搜索
1. **添加记忆**: 用户添加"我喜欢吃苹果"的记忆
2. **搜索记忆**: 搜索"我喜欢吃什么水果"
3. **验证结果**: 检查是否能找到相关的事实和对话

### 场景2: 用户代理隔离
1. **添加记忆**: 为不同用户/代理添加不同记忆
2. **搜索验证**: 确保只能搜索到自己的记忆
3. **隔离检查**: 验证不同用户间的记忆隔离

### 场景3: 相似问题召回
1. **添加问答**: 添加多个相关的问答对
2. **相似查询**: 使用不同的表达方式查询
3. **召回验证**: 检查是否能召回相关问答

### 场景4: 事实提取和搜索
1. **复杂记忆**: 添加包含多个事实的复杂记忆
2. **事实搜索**: 搜索特定的事实信息
3. **提取验证**: 检查事实提取和搜索的准确性

## 故障排除

### 常见问题

1. **搜索结果为空**
   - 检查ES和Redis服务是否正常运行
   - 确认数据已正确写入
   - 检查搜索查询是否正确

2. **用户隔离失败**
   - 检查user_id和agent_id是否正确传递
   - 确认搜索时使用了正确的过滤条件

3. **相似召回率低**
   - 检查搜索算法配置
   - 确认embedding生成正常
   - 调整相关性阈值

4. **事实提取失败**
   - 检查LLM服务是否正常
   - 确认事实提取提示词正确
   - 检查Redis连接

### 调试方法

1. **查看日志**: 检查详细的日志输出
2. **单独测试**: 运行单个测试文件进行调试
3. **数据验证**: 直接查询ES和Redis验证数据
4. **逐步调试**: 分步骤验证每个功能

## 注意事项

1. **测试数据**: 测试会创建真实的测试数据
2. **服务依赖**: 需要ES和Redis服务正常运行
3. **网络连接**: 需要LLM服务可用
4. **数据清理**: 测试完成后可以清理测试数据
